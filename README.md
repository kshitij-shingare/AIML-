# AIML- (Artificial Intelligence & Machine Learning)

## About This Repository

This repository documents a structured learning journey through **Artificial Intelligence and Machine Learning** using Python. It covers Python fundamentals first, then progressively moves into core ML concepts and models.

---

## Which Model Are We Studying?

This course covers the following **AI/ML models**, progressing from foundational to advanced:

| # | Model | Type | Description |
|---|-------|------|-------------|
| 1 | **Linear Regression** | Supervised Learning | Predict continuous output (e.g., house prices) |
| 2 | **Logistic Regression** | Supervised Learning | Binary classification (e.g., spam or not spam) |
| 3 | **Decision Tree** | Supervised Learning | Tree-based model for classification & regression |
| 4 | **Random Forest** | Ensemble Learning | Multiple decision trees for better accuracy |
| 5 | **K-Nearest Neighbors (KNN)** | Supervised Learning | Classify based on closest data points |
| 6 | **Support Vector Machine (SVM)** | Supervised Learning | Find the optimal hyperplane for classification |
| 7 | **K-Means Clustering** | Unsupervised Learning | Group data into k clusters |
| 8 | **Naive Bayes** | Supervised Learning | Probabilistic classifier based on Bayes' theorem |
| 9 | **Neural Networks (ANN)** | Deep Learning | Multi-layer perceptron for complex pattern recognition |
